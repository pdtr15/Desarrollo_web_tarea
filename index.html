Detalles de Cursos

Programación I

Puntos clave

  • Introducción a la programación con Python
  • Variables, tipos de datos y operadores
  • Estructuras de control (if, for, while)
  • Funciones y modularización
  • Introducción a POO

Evaluación

Componente Porcentaje
Exámenes teóricos 40%
Laboratorios 30%
Proyecto final 30%

Temas principales

Introducción a los algoritmos, diagramas de flujo y pseudocódigo. Primeros programas en Python. Sintaxis básica, variables, tipos de datos y operadores. Entrada y salida de datos.

Condicionales (if, else, elif), bucles (for, while) y control de flujo de programas. Estructuras anidadas y mejores prácticas en el control de flujo.

Clases, objetos, herencia, polimorfismo y encapsulamiento en Python. Diseño de programas utilizando el paradigma de orientación a objetos.

Bases de Datos

Puntos clave

  • Modelo entidad-relación
  • Normalización de bases de datos
  • Lenguaje SQL (DDL, DML)
  • Transacciones y control de concurrencia
  • Administración básica de BD

Evaluación

Componente Porcentaje
Exámenes teóricos 35%
Laboratorios prácticos 35%
Proyecto de base de datos 30%

Temas principales

Modelo entidad-relación, diagramas ER, y diseño conceptual de bases de datos. Identificación de entidades, atributos y relaciones. Cardinalidad y participación.

Creación de bases de datos, tablas, consultas SELECT, INSERT, UPDATE, DELETE, y consultas avanzadas con JOIN. Subconsultas, funciones agregadas y procedimientos almacenados.
